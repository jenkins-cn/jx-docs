## jx console

Opens the Jenkins console

### Synopsis

Opens the Jenkins X console in a browser.

```
jx console [flags]
```

### Examples

```
  # Open the Jenkins X console in a browser
  jx console
  
  # Print the Jenkins X console URL but do not open a browser
  jx console -u
```
